BBox 选框交互规范文档

适用页面：BeePOP — AI Product Studio
触发场景：状态③「AI 无法清晰识别主体」时启用

================================================================================

一、触发条件

条件                      行为
AI 自动识别成功           跳过此流程，直接进入状态②
AI 无法清晰识别主体       自动进入状态③，启用 BBox 选框工具

用户不能手动触发手动框选流程，只有 AI 判定主体不清晰时才会触发。

================================================================================

二、画布交互行为

2.1 悬停预览（Hover Preview）

触发条件：鼠标在画布空白区域移动（未选中任何框、未在拖拽操作中）
表现：以鼠标为中心，显示一个 120x120 px 的虚线预览选框（蓝色）
动态跟随：选框随鼠标实时移动
隐藏条件：
  - 鼠标悬停在已有选框上
  - 已有选框处于选中状态
  - 正在拖拽操作中
  - 鼠标离开画布区域

2.2 单击放置（Click to Place）

触发：在画布空白区域单击（非已有选框上）
行为：以点击位置为中心，放置一个 120x120 px 的实线选框
选框颜色：按顺序从调色板分配（蓝 → 紫 → 绿 → 橙 → 红 → 粉，循环）
放置后：选框立即处于「选中状态」，显示控制点和删除按钮

2.3 选中选框（Select）

触发：单击已有选框
表现：
  - 选框高亮（填充颜色加深）
  - 出现 8 个白色圆形控制点（四角 + 四边中点）
  - 右上角出现红色 × 删除按钮
  - 底部工具栏「选框大小」滑杆激活
取消选中：点击画布空白区域 或 按 Esc

2.4 移动选框（Move）

触发：在选框主体区域按住鼠标左键并拖拽
行为：选框跟随鼠标位移，松开后定位
光标变化：拖拽时光标变为 grabbing

2.5 拖拽控制点缩放（Resize via Handles）

触发：选中选框后，拖拽 8 个控制点中的任意一个
控制点方向与行为：

  控制点位置     方向标识     固定边         光标
  左上角         tl           右下角         nwse-resize
  上边中点       tc           底边           ns-resize
  右上角         tr           左下角         nesw-resize
  左边中点       ml           右边           ew-resize
  右边中点       mr           左边           ew-resize
  左下角         bl           右上角         nesw-resize
  下边中点       bc           顶边           ns-resize
  右下角         br           左上角         nwse-resize

最小尺寸限制：宽高均不得小于 32 px

2.6 通过滑杆缩放（Resize via Slider）

触发：底部工具栏「选框大小」滑杆（需有选框处于选中状态）
范围：40 px 到 280 px（对应选框宽度）
缩放逻辑：以选框中心为锚点，等比缩放（保持原始宽高比）
实时预览：拖拽滑杆时选框实时更新

2.7 删除选框（Delete）

支持三种方式：
  - 画布删除按钮：选中选框后，点击右上角红色 ×
  - 侧边栏列表：点击列表项末尾的 × 图标
  - 键盘快捷键：选中状态下按 Delete 或 Backspace

删除后，剩余选框自动重新编号（Subject 1、Subject 2…）。

================================================================================

三、底部浮动工具栏

工具栏在状态③时显示，位于画布底部中央。

  [ 选框 ]   |   选框大小  [--------o--------]

控件               状态                 说明
选框工具           始终激活             当前工具指示
选框大小滑杆       有选中框时激活       等比缩放选中框

================================================================================

四、侧边栏面板（右侧 AI Processing 区域）

状态③ 面板内容：

  AI 无法清晰识别主体
  在图像上悬停预览选框，单击放置，可添加多个

  [ 已选 2 个区域 ]

  Subject 1    120x120    [x]    <- 点击选中/高亮
  Subject 2    180x90     [x]

  [        Use this subject        ]   <- 有选框时激活

  单击放置 · 拖拽移动/缩放 · Del 删除

侧边栏与画布联动规则：
  - 点击列表项 → 画布中对应选框高亮选中
  - 画布中选中变化 → 列表对应行高亮
  - 删除选框（任何方式）→ 列表实时移除并重新编号
  - 调整选框大小 → 列表中尺寸数字实时更新

================================================================================

五、确认流程

  画布有 1 个或以上选框
       ↓
  点击「Use this subject」
       ↓
  清空所有选框和工具栏
       ↓
  进入状态④「Subject confirmed」
       ↓
  自动进入生图流程

================================================================================

六、键盘快捷键汇总

  Delete / Backspace    删除当前选中选框
  Esc                   取消选中，回到悬停预览状态

================================================================================

七、画布侧边辅助按钮

  按钮        位置                  行为
  ↺ 重置      画布左侧垂直居中      清除全部选框
  👁 预览     画布右侧垂直居中      预览（扩展功能）

================================================================================

八、多选框支持规则

  - 无数量上限，可添加任意个选框
  - 每个选框颜色独立（按顺序循环分配）
  - 选框标签自动编号：Subject 1、Subject 2…
  - 删除中间项后，所有后续标签重新编号
  - 任意时刻只能有一个选框处于「选中」状态
  - 多个选框均可单独移动、缩放、删除

================================================================================

九、交互状态流转图

  [进入状态③]
        |
        v
  [悬停预览跟随] --单击空白--> [放置选框·选中状态]
        ^                             |
        |  Esc / 点空白               +-- 拖主体 --> [移动]
        |                             +-- 拖控制点 -> [调整大小]
        |                             +-- 拖滑杆 --> [等比缩放]
        +-----------------------------+-- x/Del ---> [删除·回到预览]

  [Use this subject（1个框或以上）] --> [状态④ 确认成功]
